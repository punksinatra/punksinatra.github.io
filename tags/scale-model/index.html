Ignition Model 1:18 LB-Works Hakosuka 2Door Red

If it’s not yet obvious, I like old school cars and I like them low. Well atleast that is how I like them with toys or scale models since I do not have the means to buy one (atleast for now) beside this one is fairly hard to find nowadays.














Here is another example of how I want a hakosuka to look like if ever I had one. I really like the details on this model, rollbars, wheels, external oil cooler and the color.

Only drawback with with this model is it is resin made so the doors, hood, trunk do not open which I think is a let down for the price. Would really have looked good if those options were there.

AutoArt 1:18 Custom AE86

Have had this for quite sometime now but have always forgotten to share it here.

I have a custom AutoArt AE86 in 1/18 scale model. I like collecting weird shit (yeah for an adult to own toys is quite weird) and collecting diecast models is one of them. I really don’t go into the small models because I feel like I miss out on the details if I go with the smaller scale models.

This AE86 has been custom’d with

  • Watanabe Wheels
  • Lowered and stanced

If I had the money for an AE86, I sure would love to have it look the same as this scale model. Sharing photos and hope you like it
